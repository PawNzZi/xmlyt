# 介绍
**珠穆朗玛听** 是一款在线听书类的App。音频由后台上传，上传后可在App在线收听。由java原生开发，仅前端代码。

![](https://upyun-oos.lingyikz.cn/img/zmlm/1.png)
![](https://upyun-oos.lingyikz.cn/img/zmlm/2.png)
# 功能
- [x] 专辑分类
- [x] 专辑搜索
- [x] 快速播放
- [x] 用户注册
- [x] 用户登录
- [x] 播放历史
- [x] 专辑收藏
- [x] 在线升级
- [x] 缓存音频
- [x] 离线播放
- [x] 定时关闭

# 开发环境
- windows10
- AndroidStudio 4.2
- JDK 1.8
